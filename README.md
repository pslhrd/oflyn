# Vite + Three.JS WebGL Starter

**Boilerplate using :**

-   GSAP
-   Prettier
-   Three.JS Custom Components
-   GLSL Starter + Linting

## Setup

Make sure to install the dependencies

    yarn install

## Development

Start the development server on http://localhost:3000

```bash

yarn  dev

```

## Production

Build the application for production:

```bash

yarn build

```
